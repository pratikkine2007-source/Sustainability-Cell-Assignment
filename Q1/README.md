# Sustainability Cell Assignment

## Question 1: Hero Section Redesign

This project is a redesign of the Hero Section of the Sustainability Cell, IIT Bombay website.

The objective was to maintain the existing sustainability-focused identity while improving visual appeal, user engagement, and interactivity through modern web technologies, animations, and responsive design principles.

---

## Preview
A screenshot and a demo video of the Hero section is uploaded to the repositery, in the Q1 folder itself

---

## Features Implemented

### Visual Enhancements

* Redesigned Hero Section layout
* Improved sky-inspired background with atmospheric lighting
* Floating cloud elements
* Sustainability-focused visual theme
* Glassmorphism UI components

### Interactive Features

* 3D rotating globe
* Hover interactions on sustainability indicators
* Animated statistics counter
* Animated call-to-action buttons
* Floating leaf cursor concept

### Sustainability Indicators

Interactive sustainability metrics surrounding the globe:

* Trees Planted
* Waste Recycled
* Water Saved
* Energy Saved
* Sustainability Initiatives

Users can hover over the indicators to reveal additional information.
